* [Essential](essential/undefined.md): 개발/운영을 할 때 기본적으로 많이 사용하고 알고 있으면 유용한 내용들 (3일 과정)
* [데이터 분석](aws-ai-ml/analytics.md): AWS 에서 데이터 분석 시작을 위한 개념 잡기
* [머신러닝](undefined/aws.md): 한 시간 만에 AWS 머신러닝 이해하기
